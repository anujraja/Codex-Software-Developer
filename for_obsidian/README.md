# Obsidian Vault Folder

This folder is copy-ready for Obsidian.

## No-command setup

1. Download the repository ZIP.
2. Copy the `for_obsidian/` folder into your Obsidian vault.
3. Open `for_obsidian/00_Home.md`.

## Recommended entry points

- `00_Home.md` for navigation.
- `11_Capability_Catalog.md` for full lists.
- `graph/00_Graph_Home.md` for connected graph exploration.
