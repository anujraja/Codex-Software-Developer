# Platforms

| Platform | Best use | Notes |
|---|---|---|
| Codex CLI | Local terminal-first coding workflows | Best for repo-level editing, scripts, and command execution. |
| Codex App | Rich desktop workflows | Useful for long sessions, automations, and integrated review flows. |
| IDE Extension | In-editor coding support | Good for iterative edits inside VS Code/JetBrains/Cursor-like workflows. |
| Codex Web | Cloud task execution | Useful when local environments are constrained. |
