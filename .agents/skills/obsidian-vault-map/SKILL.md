# Obsidian Vault Map

## Use this skill when
- someone asks for a graph-based view of skills, agents, domains, or platforms
- someone wants a copy-ready folder for Obsidian
- someone needs human-readable navigation notes with wiki links

## Workflow

1. Start with one home note that links to all major views.
2. Split by platform, role/agent, skill, and domain.
3. Add one decision-guide note for "what to use when".
4. Include at least one Mermaid graph note.
5. Maintain a generated markdown inventory note for discoverability.

## Output contract

Always include:
- clear wiki-link structure
- at least one table per core note
- copy instructions for importing into Obsidian
